# Voice Profiles Directory

This directory stores voice profiles for TTS voice cloning.

## Supported Formats
- `.onnx` - Piper voice models
- `.wav` - Reference audio for voice cloning

## Adding a Piper Voice
1. Download a voice from https://github.com/rhasspy/piper/releases
2. Extract the `.onnx` file to this directory
3. Restart the backend to load the voice
